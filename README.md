# Asynchoronous Redux Action with Redux-thunk
![Redux Thunk](https://cdn.hashnode.com/res/hashnode/image/upload/v1624202557759/xnk9hl2Y3.png)

## Installation

**Create react app:**
```sh
$ npx create-react-app [my-project]
```
**Manual Setup:**
```sh
$ npm install redux redux-thunk axios
```
**Directory Structure:**
```
src
├── components
|   ├── Search
|   ├── RenderList
├── constants
├── actions
├── reducer
├── service
```
